# SimplyNode

## Setup

This repository is prepared as a Next.js + TypeScript + Tailwind project.

### 1. Install Node.js

Node.js is required to install dependencies and run the project.

- Download from https://nodejs.org/
- Or install via Homebrew if available:
  ```bash
  brew install node
  ```

### 2. Install dependencies

```bash
cd /Users/ipeople/Documents/simplynod
npm install
```

### 3. Run development server

```bash
npm run dev
```

### 4. Open in browser

Open `http://localhost:3000`

## Notes

The `components/ui` directory contains the `shape-landing-hero.tsx` component.
The `app/page.tsx` file renders this hero as the landing section.

If `npm` is still unavailable after installing Node.js, restart your terminal or add Node.js to your PATH.
